Basic Syntax in C

1. Keywords in C: The Language's Fundamental Vocabulary

Keywords are the reserved words in C with predefined meanings. They control the flow, structure, and behavior of your program.

Complete List of C Keywords

Data Type Keywords

  • int: Integer numbers
  • float: Floating-point numbers
  • double: Precise floating-point numbers
  • char: Single characters
  • void: Indicates no return value
  • short: Smaller integer range
  • long: Larger integer range
  • signed: Positive and negative values
  • unsigned: Only non-negative values

Control Flow Keywords

  • if: Conditional execution
  • else: Alternative execution path
  • switch: Multiple condition checking
  • for, while, do: Loops

Function and Memory Keywords

  • return: Send back value from function
  • const: Create unchangeable variables
  • static: Preserve variable value between calls
// Example: Keywords controlling program flow and data types
int main() {
  const float PI = 3.14159;  // 'int', 'const', 'float'
  char grade = 'A';          // 'char'
  
  if (grade == 'A') {        // 'if'
      return 0;               // 'return'
  }
  
  return 1;
}

2. Identifiers: Naming Your Programming Elements

Identifiers are the names you assign to variables, functions, and other elements. They must follow these rules:

  • Start with a letter or underscore
  • Contain letters, numbers, and underscores
  • Cannot be a keyword
  • Case-sensitive
// Valid identifiers
int student_count;
float averageScore;

// Invalid identifiers
int 2ndStudent; // Starts with a number
float total-amount; // Special character

3. Variables: Your Program's Data Containers

Variables are memory locations that store data. Here's how to declare and initialize them:

// Variable declarations
int age;
float height = 1.75;
char initial = 'J';

Types of variables:

  • Local Variables: Exist within a specific function
  • Global Variables: Accessible throughout the program
  • Static Variables: Retain value between calls

4. Data Types: Understanding Information Storage

Integer Types

int whole_number = 42;
short small_number = 100;
long big_number = 123456L;
unsigned int positive = 500;

Floating-Point Types

float decimal = 3.14f;
double precise = 3.14159;

Character Types

char letter = 'A';
char escape_char = '\n';

5. Constants: Immutable Values

Two ways to define constants:

// Using const
const float PI = 3.14159;

// Using #define
#define TAX_RATE 0.18

6. Operators: Performing Computations

Arithmetic Operators

int a = 10, b = 3;
int sum = a + b;
int prod = a * b;

Logical Operators

int p = 1, q = 0;
int and_result = p && q; // Logical AND

7. Input/Output Fundamentals

Output with printf():

int age = 25;
printf("Age: %d", age);

Input with scanf():

int user_age;
scanf("%d", &user_age);
