Getting Started

Downloading Linux

    1. Choose a Linux distribution: There are many different Linux distributions available, each with its own set of features and characteristics. Some of the most popular distributions include Ubuntu, Debian, Fedora, and CentOS. You'll want to choose a distribution that meets your needs and matches your level of experience.

    2. Download the distribution: Once you've chosen a distribution, you'll need to download the installation files. Most distributions can be downloaded for free from their respective websites.

    3. Create installation media: After downloading the installation files, you'll need to create installation media. This can be done by burning the files to a DVD or USB drive using software like Rufus or Etcher.

    4. Boot from the installation media: With the installation media ready, you'll need to boot your computer from it. This can usually be done by changing the boot order in your computer's BIOS or UEFI settings.

    5. Install Linux: Once you've booted from the installation media, you'll be presented with the option to install Linux. Follow the on-screen prompts to complete the installation process.

    6. Set up your system: After installing Linux, you'll need to set up your system. This may involve configuring your network settings, installing additional software, and customizing your desktop environment.

    7. Learn the basics: With Linux installed and configured, you'll want to start learning the basics. This includes learning how to navigate the file system, using the command line interface, and installing and managing software.

    8. Join the community: Linux has a large and active community of users and developers who can provide help and support as you continue to learn and use the operating system. Joining forums, mailing lists, and IRC channels can be a great way to connect with others and get help when you need it.

Accessing the CLI

  1. Open the Terminal: The Terminal is the default command line interface in Linux. You can open it by clicking on the Terminal icon in the Applications menu, or by pressing the ctrl + alt + T keyboard shortcut.

  2. Use the keyboard: Once you've opened the Terminal, you can use the keyboard to enter commands and navigate the file system

Basic Commands and Syntax

  1. pwd - Print Working Directory: Displays the path of the current directory. Syntax: pwd

  2. cd - Change Directory: Allows you to change the current directory. Syntax: cd [directory name]

  3. ls - List: Lists the contents of the current directory. Syntax: ls [options] [directory name]

  4. mkdir - Make Directory: Creates a new directory. Syntax: mkdir [directory name]

  5. rmdir - Remove Directory: Deletes an empty directory Syntax: rmdir [directory name]

  6. touch - Create File: Creates a new empty file. Syntax: touch [filename]

  7. cp - Copy: Copies files or directories. Syntax: cp [options] [source file/directory] [destination file/directory]

  8. rm - Remove: Deletes files or directories. Syntax: rm [options] [filename/directory name]

  9. mv - Move: Moves or renames files or directories. Syntax: mv [options] [source file/directory] [destination file/directory]

  10. cat - Concatenate: Displays the contents of a file. Syntax: cat [filename]

Note: Be careful when using the rm command, as it can permanently delete files and directories without confirmation. Always double-check the files and directories you're deleting before running the command.
